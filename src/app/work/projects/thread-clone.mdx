## Overview

Thread Clone is a social media platform that allows users to post updates, follow others, and engage in conversations. It focuses on a clean user interface and seamless user experience.

## Key Features

- **User Authentication**: Secure login and signup using JWT.
- **Post Creation**: Users can create, edit, and delete their threads.
- **Interactions**: Like and comment on threads.
- **User Profiles**: Customizable profiles with follow/unfollow functionality.

## Tech Stack

- **React**: Frontend library for building the UI.
- **Node.js**: Backend runtime environment.
- **Express**: Web framework for the API.
- **MongoDB**: NoSQL database for storing user and post data.
- **Tailwind CSS**: For modern and responsive styling.
