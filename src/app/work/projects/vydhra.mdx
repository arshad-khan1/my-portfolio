## Overview

Vydhra is a cutting-edge online education platform designed to democratize learning. It provides a seamless interface for students to access a vast library of courses, track their progress, and engage with interactive learning materials.

## Key Features

- **Course Management**: Intuitive dashboard for students to manage their enrolled courses and certifications.
- **Dynamic Learning**: Support for various media formats, including video lectures, interactive quizzes, and downloadable resources.
- **Progress Tracking**: Real-time monitoring of learning milestones and achievement badges.
- **Responsive Interface**: Optimized for all devices, ensuring a consistent learning experience on desktop, tablet, and mobile.

## Tech Stack

- **Next.js**: For a high-performance, SEO-friendly frontend and API routes.
- **Tailwind CSS**: For a sleek, modern, and responsive design system.
- **TypeScript**: Ensuring type safety and robust codebase.
