## Overview

The Gym Management Application is a cross-platform mobile app designed to simplify the day-to-day operations of a gym. It provides tools for member management, attendance tracking, and staff scheduling.

## My Contribution

- **Full-Stack Setup**: Initialized both backend and frontend projects with a scalable folder structure.
- **Database Design**: Designed and implemented the MongoDB schema for members, trainers, and gym data.
- **Role-Based Auth**: Implemented a secure authentication system with JWT and protected routes for different user roles.
- **API Development**: Developed the core RESTful APIs required for the mobile application.

## Tech Stack

- **React Native Expo**: For building the cross-platform mobile application.
- **Node.js**: Backend runtime environment.
- **MongoDB**: NoSQL database for flexible data storage.
- **JWT**: For secure, token-based authentication.
