## Overview

BeanBagAffairs is a visually-driven website designed to showcase the portfolio of a creative agency specializing in Ad Films and Wedding Photography. It focuses on high-quality media delivery and a premium user experience.

## Key Features

- **Portfolio Showcase**: High-resolution image and video galleries.
- **Service Listings**: Detailed descriptions of offered services.
- **Contact Integration**: Easy-to-use contact forms for potential clients.
- **Optimized Media**: Uses Cloudinary for fast and efficient media delivery.

## Tech Stack

- **Next.js**: For a fast, SEO-friendly frontend.
- **Cloudinary CDN**: For high-performance image and video hosting.
- **Tailwind CSS**: For a sleek and modern design.
