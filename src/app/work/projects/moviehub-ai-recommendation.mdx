## Overview

MovieHub is a movie browsing platform that leverages the TMDB API to provide users with comprehensive movie data. Integrated with AI for recommendations, it offers dynamic search and filtering capabilities to enhance the user experience.

## Key Features

- **Movie Data Integration**: Utilized TMDB API to fetch detailed movie information including ratings, cast, reviews, and streaming platforms.
- **Dynamic Listings**: Built searchable and filterable movie listings with options to filter by genre, rating, and release date.
- **Detailed Pages**: Created movie detail pages displaying cast, reviews, and available streaming platforms for a rich user experience.
- **AI Recommendations**: Incorporated TensorFlow and PyTorch for AI-driven movie recommendations based on user preferences.

## Technologies Used

- **React**: For building the interactive user interface.
- **TMDB API**: For fetching movie data.
- **Tailwind CSS**: For styling the application.
- **JavaScript**: As the primary programming language.
- **TensorFlow and PyTorch**: For implementing AI recommendation algorithms.

## Challenges and Learnings

Integrating AI recommendations required processing large datasets and optimizing algorithms for real-time suggestions. I learned to balance performance with accuracy in machine learning models.

## Outcome

MovieHub provides an engaging platform for movie enthusiasts, combining API integration with AI to deliver personalized recommendations and detailed movie insights.
