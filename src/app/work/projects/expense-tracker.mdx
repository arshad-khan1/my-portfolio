## Overview

Expense Tracker is a high-performance backend system designed to manage personal finances. It leverages a modern microservices-ready architecture to handle real-time data and event-driven processes.

## Key Features

- **Live Balances**: Real-time balance tracking using Redis caching.
- **Bank Sync**: Automated synchronization with bank accounts (simulated via Kafka).
- **Category Management**: Intelligent categorization of expenses with caching for performance.
- **Event-Driven Architecture**: Uses Kafka for decoupling services and handling asynchronous tasks.

## Tech Stack

- **NestJS**: Framework for building efficient and scalable server-side applications.
- **Prisma**: Type-safe ORM for PostgreSQL.
- **PostgreSQL**: Relational database for persistent storage.
- **Redis**: In-memory data store for live balances and caching.
- **Kafka**: Distributed event streaming platform for bank-sync and downstream services.
