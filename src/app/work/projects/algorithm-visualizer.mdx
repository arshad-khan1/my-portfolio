## Overview

The Algorithm Visualizer is an educational tool designed to help users understand sorting and pathfinding algorithms through interactive visualizations. Users can adjust input sizes, animation speeds, and observe the algorithms in action.

## Key Features

- **Sorting Algorithms**: Implemented visualizations for algorithms like Bubble Sort, with step-by-step animations.
- **Pathfinding Algorithms**: Added support for A\* and Dijkstra's algorithms, showing path exploration and optimization.
- **Interactive Controls**: Enabled users to modify input sizes, animation speed, and interact with the visualizer for real-time observation.

## Technologies Used

- **React**: For creating the dynamic and interactive interface.
- **CSS**: For styling the visualizations and animations.
- **JavaScript**: For implementing the algorithm logic and animations.

## Challenges and Learnings

Creating smooth animations while maintaining performance for large inputs was challenging. I optimized the rendering process and used efficient data structures to handle complex visualizations.

## Outcome

The Algorithm Visualizer serves as an effective learning tool, making abstract algorithms tangible and easier to comprehend through visual representation.
