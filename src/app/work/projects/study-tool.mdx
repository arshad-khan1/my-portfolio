## Overview

Study Tool is a comprehensive flashcard application designed to help students and professionals learn more effectively. It uses advanced spaced repetition algorithms to schedule card reviews at optimal intervals.

## My Contribution

- **User Dashboard**: Implemented the main dashboard for users to track their progress.
- **Theme Support**: Added dark and light mode support for better user experience.
- **Authentication**: Built the authentication system for both frontend and backend.
- **SM-15 Algorithm**: Implemented the SM-15 algorithm to optimize the learning curve and card repetition.

## Tech Stack

### Backend

- **NestJS**: Framework for building the API.
- **PostgreSQL**: Relational database with Prisma ORM.
- **JWT**: For secure authentication.
- **Docker**: For containerized deployment.

### Frontend

- **Next.js**: React framework for the user interface.
- **Tailwind CSS**: For styling with shadcn components.
- **React Context**: For state management.
