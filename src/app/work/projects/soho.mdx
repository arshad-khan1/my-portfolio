## Overview

Soho is a premium ecommerce solution consisting of a high-performance mobile application and a powerful administrative dashboard. It offers a sophisticated shopping experience with a focus on usability, security, and elegant design.

## Key Features

- **Intuitive Shopping**: Seamless product discovery with advanced filtering and search capabilities.
- **Secure Checkout**: Robust and secure payment integration for a safe shopping experience.
- **Admin Dashboard**: A comprehensive management suite for products, orders, customers, and analytics.
- **Push Notifications**: Real-time updates for orders, promotions, and personalized recommendations.
- **User Accounts**: Detailed user profiles with order history and wishlist functionality.

## Tech Stack

- **React Native**: For a native mobile experience on both iOS and Android.
- **Node.js**: Powering the scalable backend API.
- **PostgreSQL**: Reliable relational database for managing products and user data.
- **Redis**: For high-performance caching and session management.

## Admin Dashboard

The Soho project includes a custom-built **Admin Dashboard** that allows administrators to:

- Manage product inventory and pricing.
- Track real-time sales and customer analytics.
- Process and manage orders efficiently.
- Handle customer support and feedback.
